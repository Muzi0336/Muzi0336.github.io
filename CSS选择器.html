CSS 选择器

一、元素选择器

概念：用 HTML 标签名直接选择元素，页面中所有该标签都会被选中。
语法：标签名 { 属性: 值; }
👇 演示：下面所有 p 标签都是红色

我是第一个段落

我是第二个段落

我是第三个段落

/* CSS */ p { color: red; font-size: 16px; } <!-- HTML --> <p>我是第一个段落</p> <p>我是第二个段落</p> <p>我是第三个段落</p>

二、类选择器

概念：用 .类名 选择所有拥有该 class 的元素，一个元素可以有多个类。
语法：.类名 { 属性: 值; }
👇 演示：橙色加粗 = .highlight，蓝色 = .text-blue

我是 highlight，橙色加粗

我是 text-blue，蓝色

我同时有两个类

/* CSS */ .highlight { color: orange; font-weight: bold; } .text-blue { color: blue; } <!-- HTML --> <p class="highlight">橙色加粗</p> <p class="text-blue">蓝色</p> <p class="highlight text-blue">同时有两个类</p>

三、ID 选择器

概念：用 #ID名 选择唯一一个元素，页面中 ID 必须唯一。
语法：#ID名 { 属性: 值; }
👇 演示：紫色加大 = #special

我是 special，紫色加大

我是普通段落

/* CSS */ #special { color: purple; font-size: 18px; } <!-- HTML --> <p id="special">紫色加大</p> <p>普通段落</p>

四、通配符选择器

概念：用 * 选择页面中所有元素，常用于重置默认样式。
语法：* { 属性: 值; }
注意：性能消耗较大，实际开发中一般只用于 margin/padding 重置。
👇 演示：页面所有元素的 margin 和 padding 已被清零（* { margin: 0; padding: 0; }）

这个段落紧贴容器边缘，没有默认间距

/* CSS */ * { margin: 0; padding: 0; box-sizing: border-box; }

五、后代选择器

概念：用 祖先 后代（空格）选择某元素内部所有层级的后代。
语法：父元素 后代元素 { 属性: 值; }
👇 演示：.ancestor 内所有 span 都是绿色（包括嵌套多层的）
直接子元素 span → 绿色
嵌套在 div 里的 span → 也是绿色
/* CSS */ .ancestor span { color: green; font-weight: bold; } <!-- HTML --> <div class="ancestor"> <span>直接子元素 → 绿色</span> <div> <span>嵌套多层 → 也是绿色</span> </div> </div>

六、子元素选择器

概念：用 父 > 子（大于号）只选择直接子元素，不选更深层的后代。
语法：父元素 > 子元素 { 属性: 值; }
对比：后代选择器选所有层级，子元素选择器只选第一层。
👇 演示：只有 .parent 的直接子 p 是红色，嵌套的不受影响

直接子元素 p → 红色

嵌套在 div 里的 p → 不受影响

/* CSS */ .parent > p { color: red; } <!-- HTML --> <div class="parent"> <p>直接子元素 → 红色</p> <div> <p>嵌套的 p → 不受影响</p> </div> </div>

七、相邻兄弟选择器

概念：用 A + B（加号）选择紧跟在 A 后面的第一个 B 元素。
语法：A + B { 属性: 值; }
关键：必须是紧挨着的下一个，中间不能有其他元素。
👇 演示：紧跟 h3 后面的 p 是紫色加粗

标题

紧跟 h3 的 p → 紫色加粗

第二个 p → 普通样式

/* CSS */ h3 + p { color: purple; font-weight: bold; } <!-- HTML --> <h3>标题</h3> <p>紧跟 h3 → 紫色加粗</p> <p>第二个 p → 普通</p>

八、通用兄弟选择器

概念：用 A ~ B（波浪号）选择 A 后面所有同级的 B 元素。
语法：A ~ B { 属性: 值; }
对比：相邻兄弟只选第一个，通用兄弟选后面所有。
👇 演示：h3 后面所有 p 都是蓝色

h3 前面的 p → 普通

标题

h3 后面第一个 p → 蓝色

h3 后面第二个 p → 蓝色

/* CSS */ h3 ~ p { color: blue; } <!-- HTML --> <p>h3 前面 → 普通</p> <h3>标题</h3> <p>后面第一个 → 蓝色</p> <p>后面第二个 → 蓝色</p>

九、分组选择器

概念：用 A, B（逗号）同时选中多个元素，给它们设置相同样式。
语法：A, B { 属性: 值; }
作用：减少重复代码，一次定义多个元素的共同样式。
👇 演示：h3 和 p 都是橙色

我是标题

我是段落

/* CSS */ h3, p { color: orange; } <!-- HTML --> <h3>我是标题</h3> <p>我是段落</p>

十、属性选择器

概念：根据元素的属性来选择元素，有多种匹配方式：
[title] — 有 title 属性
[type="text"] — type 等于 text
[href^="https"] — href 以 https 开头
[href$=".pdf"] — href 以 .pdf 结尾
[href*="github"] — href 包含 github
👇 演示：不同属性匹配不同样式

有 title → 红色边框

https 开头 → 绿色加粗 .pdf 结尾 → 红色 包含 github → 紫色下划线
/* CSS */ [title] { border: 2px solid red; } [type="text"] { border: 2px solid blue; background: lightblue; } [href^="https"] { color: green; font-weight: bold; } [href$=".pdf"] { color: red; } [href*="github"] { color: purple; text-decoration: underline; } <!-- HTML --> <p title="提示信息">有 title 的段落</p> <input type="text"> <a href="https://github.com">https 开头</a> <a href="file.pdf">.pdf 结尾</a> <a href="https://github.com/repo">包含 github</a>

十一、:hover 伪类

概念：鼠标悬停在元素上时触发的样式。
语法：元素:hover { 属性: 值; }
常用场景：按钮高亮、链接变色、卡片浮起效果。
👇 演示：鼠标放上去试试
鼠标悬停到这里 → 蓝色背景 + 放大
/* CSS */ .box:hover { background: #3498db; color: white; transform: scale(1.02); } <!-- HTML --> <div class="box">鼠标悬停到这里</div>

十二、:active 伪类

概念：鼠标按下（点击瞬间）时触发的样式。
语法：元素:active { 属性: 值; }
常用场景：按钮按下反馈、点击变色。
👇 演示：按住鼠标不放试试
按住我 → 红色背景 + 缩小
/* CSS */ .box:active { background: #e74c3c; color: white; transform: scale(0.98); } <!-- HTML --> <div class="box">按住我</div>

十三、:focus 伪类

概念：元素获得焦点时触发的样式，主要用于表单元素。
语法：元素:focus { 属性: 值; }
常用场景：输入框聚焦高亮、表单验证提示。
👇 演示：点击输入框获得焦点
/* CSS */ input:focus { border-color: #3498db; box-shadow: 0 0 8px rgba(52,152,219,0.3); } <!-- HTML --> <input type="text" placeholder="点我获得焦点">

十四、:first-child / :last-child

概念：:first-child 选第一个子元素，:last-child 选最后一个子元素。
语法：元素:first-child { ... } / 元素:last-child { ... }
👇 演示：第一个 li 红色，最后一个 li 蓝色
  • 第 1 个 → 红色加粗（:first-child）
  • 第 2 个
  • 第 3 个
  • 第 4 个 → 蓝色加粗（:last-child）
/* CSS */ li:first-child { color: red; font-weight: bold; } li:last-child { color: blue; font-weight: bold; } <!-- HTML --> <ul> <li>第 1 个 → 红色</li> <li>第 2 个</li> <li>第 3 个</li> <li>第 4 个 → 蓝色</li> </ul>

十五、:nth-child

概念：按位置选择子元素，非常灵活。
:nth-child(odd) — 奇数项（1, 3, 5...）
:nth-child(even) — 偶数项（2, 4, 6...）
:nth-child(n) — 第 n 个
常用场景：斑马纹表格、隔行变色。
👇 演示：奇数行蓝底，偶数行橙底，第 3 个红色加粗
  • 第 1 个（奇数）→ 蓝底
  • 第 2 个（偶数）→ 橙底
  • 第 3 个（:nth-child(3)）→ 红色加粗
  • 第 4 个（偶数）→ 橙底
  • 第 5 个（奇数）→ 蓝底
/* CSS */ li:nth-child(odd) { background: #ebf5fb; } li:nth-child(even) { background: #fdebd0; } li:nth-child(3) { color: red; font-weight: bold; } <!-- HTML --> <ul> <li>第 1 个（奇数）</li> <li>第 2 个（偶数）</li> <li>第 3 个</li> ... </ul>

十六、:not 伪类

概念：排除符合条件的元素，选中"除了这个以外"的所有元素。
语法：元素:not(.类名) { ... }
常用场景：给所有元素加样式，但排除某个特殊的。
👇 演示：所有 p 都是绿色加粗，除了 .skip

我是普通 p → 绿色加粗

我是 .skip → 灰色（被排除）

我也是普通 p → 绿色加粗

/* CSS */ p { color: gray; } p:not(.skip) { color: green; font-weight: bold; } <!-- HTML --> <p>普通 p → 绿色</p> <p class="skip">.skip → 灰色</p> <p>普通 p → 绿色</p>

十七、::before / ::after 伪元素

概念：在元素内容的前面/后面插入额外内容，必须配合 content 属性使用。
语法：元素::before { content: "..."; }
常用场景：添加图标、装饰符号、计数器。
👇 演示：每个 p 前后自动添加了符号

我是段落内容

我也是段落内容

/* CSS */ p::before { content: "👉 "; } p::after { content: " 👈"; } <!-- HTML --> <p>我是段落内容</p> <!-- 渲染结果：👉 我是段落内容 👈 -->

十八、::first-letter 伪元素

概念：选中元素的第一个字符，可以单独设置样式。
语法：p::first-letter { ... }
常用场景：段首大字效果（类似报纸排版）。
👇 演示：第一个字是红色大号

hello world，这是第一个字符的演示效果。

/* CSS */ p::first-letter { font-size: 32px; color: red; font-weight: bold; } <!-- HTML --> <p>hello world</p> <!-- 渲染结果：第一个字母 h 变成红色大号 -->

十九、::first-line 伪元素

概念：选中元素的第一行文字，根据浏览器窗口宽度动态变化。
语法：p::first-line { ... }
注意：只能用于块级元素，且只对文字生效。
👇 演示：第一行是蓝色加粗（缩放窗口会变化）

这是一段比较长的文字，用来演示 ::first-line 的效果。无论窗口多宽，只有第一行会变成蓝色加粗。当你改变浏览器窗口宽度时，第一行的内容会动态变化，蓝色加粗的范围也会跟着变。

/* CSS */ p::first-line { color: blue; font-weight: bold; } <!-- HTML --> <p>这是一段比较长的文字...</p> <!-- 只有第一行变蓝，窗口宽度变化时范围动态调整 -->

二十、::selection 伪元素

概念：选中文字被高亮选中时的样式。
语法：p::selection { ... }
常用场景：自定义选中文字的背景色和文字色。
👇 演示：用鼠标选中下面的文字，背景是黄色

用鼠标选中我试试，选中后的背景色是黄色，文字是深色。

/* CSS */ p::selection { background: #f1c40f; color: #2c3e50; } <!-- HTML --> <p>选中我试试</p>

二十一、:root 伪类

概念：:root 选中根元素（即 html），常用于定义 CSS 变量。
语法：:root { --变量名: 值; }
常用场景：全局主题色、统一管理颜色/间距。
👇 演示：用 CSS 变量控制颜色

这段文字的颜色由 :root 定义的 --main-color 控制

/* CSS */ :root { --main-color: #3498db; } p { color: var(--main-color); } <!-- 修改 :root 中的变量，所有使用它的地方都会变 -->

二十二、:empty 伪类

概念：选中没有任何内容的元素（包括子元素和文本）。
语法：元素:empty { ... }
常用场景：空容器提示、表单空值警告。
👇 演示：空的 div 有红色背景提示
我有内容，正常显示
我也有内容
/* CSS */ div:empty { background: #fadbd8; } div:empty::after { content: "这个 div 是空的"; color: red; } <!-- HTML --> <div>有内容</div> <div></div> <!-- 空的，会变红 --> <div>有内容</div> <div></div> <!-- 空的，会变红 -->

二十三、:checked 伪类

概念：选中被勾选的复选框或单选按钮。
语法：input:checked + label { ... }
常用场景：自定义复选框样式、勾选后文字变化。
👇 演示：勾选后文字变绿并加删除线


/* CSS */ input[type="checkbox"]:checked + label { color: green; font-weight: bold; text-decoration: line-through; } <!-- HTML --> <input type="checkbox" id="c1"> <label for="c1">学习 HTML</label>

二十四、:disabled / :enabled

概念：:disabled 选中被禁用的元素，:enabled 选中可用的元素。
语法：input:disabled { ... }
👇 演示：禁用的输入框是灰色，可用的是蓝色边框
/* CSS */ input:disabled { background: #ecf0f1; cursor: not-allowed; } input:enabled { border: 2px solid blue; } <!-- HTML --> <input type="text" value="可用"> <input type="text" value="禁用" disabled>

二十五、:valid / :invalid

概念：:valid 选中输入内容合法的表单元素，:invalid 选中不合法的。
语法：input:valid { ... } / input:invalid { ... }
常用场景：表单实时验证、输入框边框变色提示。
👇 演示：输入邮箱，合法绿色边框，不合法红色边框
/* CSS */ input:valid { border: 2px solid green; } input:invalid { border: 2px solid red; } <!-- HTML --> <input type="email" placeholder="输入邮箱"> <!-- 输入 abc → 红色（不合法）--> <!-- 输入 abc@123.com → 绿色（合法）-->

二十六、:only-child 伪类

概念：选中是其父元素唯一子元素的元素。
语法：元素:only-child { ... }
常用场景：单独一个子元素时特殊处理。
👇 演示：只有一个 li 的 ul 会被特殊标记
  • 我是唯一的 li → 红色加粗（:only-child）
  • 我有兄弟
  • 我也有兄弟
/* CSS */ li:only-child { color: red; font-weight: bold; background: #fadbd8; } <!-- HTML --> <ul> <li>唯一的 li → 红色</li> </ul> <ul> <li>有兄弟</li> <li>也有兄弟</li> </ul>

二十七、:first-of-type / :last-of-type

概念：:first-of-type 选中父元素中第一个同类型的子元素，:last-of-type 选中最后一个。
语法：p:first-of-type { ... }
对比：:first-child 要求是第一个子元素，:first-of-type 只要是第一个 p 就行，前面可以有其他标签。
👇 演示：第一个 p 绿色，最后一个 p 紫色

这是标题

第一个 p → 绿色加粗（:first-of-type）

第二个 p

第三个 p → 紫色加粗（:last-of-type）

/* CSS */ p:first-of-type { color: green; font-weight: bold; } p:last-of-type { color: purple; font-weight: bold; } <!-- HTML --> <h3>标题</h3> <p>第一个 p → 绿色</p> <p>第二个 p</p> <p>第三个 p → 紫色</p>

二十八、:nth-of-type

概念：按类型选择子元素，只计算同类型的元素。
:nth-of-type(odd) — 同类型中的奇数项
:nth-of-type(even) — 同类型中的偶数项
对比：:nth-child 计算所有子元素，:nth-of-type 只计算同类型。
👇 演示：p 标签奇数行蓝底，偶数行橙底

标题不参与计算

第一个 p（奇数）→ 蓝底

第二个 p（偶数）→ 橙底

第三个 p（奇数）→ 蓝底

/* CSS */ p:nth-of-type(odd) { background: #ebf5fb; } p:nth-of-type(even) { background: #fdebd0; } <!-- HTML --> <h3>标题不参与计算</h3> <p>第一个 p（奇数）</p> <p>第二个 p（偶数）</p> <p>第三个 p（奇数）</p>

二十九、:required / :optional

概念：:required 选中必填的表单元素，:optional 选中选填的。
语法：input:required { ... }
判断依据：有 required 属性就是必填，没有就是选填。
👇 演示：必填红色边框，选填绿色边框
/* CSS */ input:required { border: 2px solid red; } input:optional { border: 2px solid green; } <!-- HTML --> <input type="text" required> <!-- 必填 → 红色 --> <input type="text"> <!-- 选填 → 绿色 -->

三十、选择器优先级

概念：当多个选择器选中同一个元素时，按优先级决定谁生效。
优先级从低到高：
1. 元素选择器（p）→ 权重 1
2. 类选择器（.class）→ 权重 10
3. ID 选择器（#id）→ 权重 100
4. inline 样式 → 权重 1000
口诀：ID > 类 > 元素，权重高的覆盖权重低的。
👇 演示：同一个 p 元素被三个选择器选中，最终 #id 生效

我是同一个 p，最终显示红色加粗（ID 优先级最高）

/* CSS — 三个选择器都选中了这个 p */ p { color: gray; } /* 权重 1 */ .specific { color: blue; } /* 权重 10 */ #very-specific { color: red; font-weight: bold; } /* 权重 100 ✓ */ <!-- HTML --> <p id="very-specific" class="specific">最终显示红色</p> <!-- 如果要强制覆盖，可以用 !important（但不推荐滥用）--> /* p { color: gray !important; } */
