# Notes Learning Repository

This is a personal learning notes repository documenting my studies on Python and FastAPI.

## Overview

### Python Notes (python.md)

Learning notes on Python fundamentals, covering:

- **Basic Syntax**: Variables, data types, strings, lists, tuples, sets, dictionaries
- **Functions**: Positional arguments, keyword arguments, variable-length arguments (*args, **kwargs)
- **Common Modules**: json, math, os, datetime, re, random, csv, collections, functools
- **Object-Oriented Programming**: Class definition, inheritance, magic methods, attribute access control
- **Advanced Features**: Decorators, generators, iterators, context managers, closures
- **Type Hints**: Various uses of Type Hints
- **Coroutines**: Basic async/await asynchronous programming

### FastAPI Notes (fastAPI.md)

Learning notes on the FastAPI framework, covering:

- **Quick Start**: Creating an instance, root path, hello page
- **Data Models**: Defining request and response models with Pydantic
- **Route Registration**: Configuring routes for modules such as articles and activities
- **Database Operations**: Asynchronous SQLAlchemy operations, relationship mapping, transaction management
- **Authentication & Authorization**: JWT tokens, JWT validation, custom dependency injection
- **Middleware**: CORS configuration, request logging, rate limiting logic
- **Exception Handling**: Custom exceptions, global exception handlers
- **File Upload**: File storage and static file access
- **Configuration Management**: Environment variables, global singleton pattern
- **Pagination & Filtering**: Generic pagination dependency encapsulation

## Usage Instructions

These notes are compiled during the learning process and are suitable as reference material for review. The Markdown format makes them easy to read and edit.
