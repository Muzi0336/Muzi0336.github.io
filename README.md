

# 📚 Note 学习笔记

这是一个个人学习笔记仓库，记录 Python 和 FastAPI 相关学习内容。

## 📖 内容简介

本仓库收录了 Python 基础知识及 FastAPI Web 框架的系统学习笔记，适合作为复习参考资料。

### 🐍 Python 笔记 (`python.md`)

| 模块 | 主要内容 |
|------|----------|
| **基础语法** | 变量、数据类型、字符串、列表、元组、集合、字典 |
| **函数** | 位置参数、关键字参数、不定长参数 (*args, **kwargs) |
| **常用模块** | json, math, os, datetime, re, random, csv, collections, functools |
| **面向对象** | 类定义、继承、魔术方法、属性访问控制 |
| **高级特性** | 装饰器、生成器、迭代器、上下文管理器、闭包 |
| **类型提示** | Type Hints 各式用法 |
| **协程** | async/await 异步编程基础 |

### ⚡ FastAPI 笔记 (`fastAPI.md`)

| 模块 | 主要内容 |
|------|----------|
| **快速入门** | 创建实例、根目录、hello 页面 |
| **数据模型** | Pydantic 请求体与响应体模型定义 |
| **路由注册** | 活动模块、文章模块等路由配置 |
| **数据库操作** | SQLAlchemy 异步操作、关联关系、事务管理 |
| **认证授权** | JWT Token 生成与验证、自定义依赖注入 |
| **中间件** | CORS 配置、请求日志、限流逻辑 |
| **异常处理** | 自定义异常、全局异常处理器 |
| **文件上传** | 文件保存与静态文件访问 |
| **配置管理** | 环境变量、全局单例模式 |
| **分页与筛选** | 通用分页依赖封装 |
