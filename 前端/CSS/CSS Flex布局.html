🎯 CSS Flex布局

1. Flex 基础概念

Flexbox 是一维布局的王牌，用于在一行或一列中排列元素。

  • 容器：设置 display: flex 的元素
  • 项目：容器的直接子元素
  • 主轴：项目排列的方向（默认水平）
  • 交叉轴：与主轴垂直的方向（默认垂直）
👇 最简单的 Flex 布局
1
2
3
/* 容器设置 */ .container { display: flex; gap: 10px; } /* 项目会自动排列在一行 */

2. flex-direction（主轴方向）

flex-direction 定义主轴方向，决定项目如何排列。

align-items 控制项目在交叉轴上的对齐方式。

5. gap（间距）

gap 设置项目之间的间距，比 margin 更方便。

👇 gap 演示
1
2
3
/* 统一间距 */ .container { gap: 20px; } /* 行列分别设置 */ .container { row-gap: 20px; column-gap: 10px; }

6. flex-wrap（换行）

flex-wrap 控制项目是否换行。

默认不换行，项目会压缩以适应容器。
