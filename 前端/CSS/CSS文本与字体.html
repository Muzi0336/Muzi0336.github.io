📝 CSS文本与字体

1. font-size（字体大小）

font-size 设置文字大小。

推荐单位：rem（相对于根元素）或 px（绝对单位）。

👇 不同字体大小

12px - 小号文字

16px - 正文（默认）

20px - 小标题

24px - 标题

1.5rem - 使用rem单位

/* px 绝对单位 */ font-size: 16px; /* rem 相对根元素（推荐） */ font-size: 1rem; /* em 相对父元素 */ font-size: 1.2em; /* 百分比 */ font-size: 120%;

2. font-weight（字体粗细）

font-weight 设置文字粗细。

常用值：normal(400)、bold(700)。

👇 不同字体粗细

300 - 细体

400 - 正常（normal）

500 - 中等

700 - 粗体（bold）

900 - 特粗

/* 关键字 */ font-weight: normal; font-weight: bold; /* 数值（100-900） */ font-weight: 400; /* 等同 normal */ font-weight: 700; /* 等同 bold */

3. font-family（字体族）

font-family 设置字体，可以指定多个备选字体。

中文字体名需要加引号。

👇 不同字体

微软雅黑 - Microsoft YaHei

宋体 - SimSun

楷体 - KaiTi

Arial 无衬线字体

Times New Roman 衬线字体

/* 单个字体 */ font-family: 'Microsoft YaHei'; /* 多个备选（从左到右优先级） */ font-family: 'Microsoft YaHei', 'SimSun', sans-serif; /* 常见字体分类 */ /* 无衬线：Arial, Helvetica, sans-serif */ /* 衬线：Times New Roman, serif */ /* 等宽：Consolas, monospace */

4. line-height（行高）

line-height 设置行与行之间的距离。

常用值：1.5-2（无单位，相对于字体大小）。

👇 不同行高对比

line-height: 1
这是一段测试文字
看看行高效果

line-height: 1.5
这是一段测试文字
看看行高效果

line-height: 2
这是一段测试文字
看看行高效果

/* 无单位（推荐，相对于字体大小） */ line-height: 1.5; /* 像素 */ line-height: 24px; /* 百分比 */ line-height: 150%; /* 垂直居中技巧 */ .line-height-center { line-height: 40px; /* 等于容器高度 */ height: 40px; }

5. color（文字颜色）

color 设置文字颜色。

👇 不同颜色表示法

关键字：red

十六进制：#3498db

RGB：rgb(46, 204, 113)

RGBA（带透明度）：rgba(241, 196, 15, 0.8)

/* 关键字 */ color: red; /* 十六进制 */ color: #3498db; color: #398; /* 简写 */ /* RGB */ color: rgb(52, 152, 219); /* RGBA（带透明度） */ color: rgba(52, 152, 219, 0.8);

6. text-align（文本对齐）

text-align 控制文本的水平对齐方式。

👇 不同对齐方式
text-align: left 左对齐
text-align: center 居中
text-align: right 右对齐
text-align: justify 两端对齐，这是一段较长的文字用来演示两端对齐的效果
/* 左对齐（默认） */ text-align: left; /* 居中 */ text-align: center; /* 右对齐 */ text-align: right; /* 两端对齐 */ text-align: justify;

7. text-decoration（文本装饰）

text-decoration 添加下划线、删除线等装饰。

👇 不同装饰效果

text-decoration: none（无装饰）

text-decoration: underline（下划线）

text-decoration: overline（上划线）

text-decoration: line-through（删除线）

/* 无装饰（常用于去除链接下划线） */ text-decoration: none; /* 下划线 */ text-decoration: underline; /* 上划线 */ text-decoration: overline; /* 删除线 */ text-decoration: line-through; /* 组合 */ text-decoration: underline wavy red;
💡 font 复合属性：可以一行设置多个字体属性：font: bold 16px/1.5 'Microsoft YaHei';（粗细 字号/行高 字体）
