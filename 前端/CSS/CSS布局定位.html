📐 CSS布局定位

1. display（显示模式）

display 是布局的核心属性，决定元素的显示方式。

  • block：块级元素，独占一行
  • inline：行内元素，不能设置宽高
  • inline-block：行内块，既能设置宽高又不独占一行
  • flex：弹性布局
  • grid：网格布局
  • none：隐藏元素
👇 display 不同值对比
block 块级元素（独占一行）
inline 行内元素
inline 行内元素
inline-block 行内块
inline-block 行内块
/* 块级元素 */ div { display: block; } /* 行内元素 */ span { display: inline; } /* 行内块 */ a { display: inline-block; width: 100px; } /* 隐藏元素 */ .hidden { display: none; }

2. position（定位方式）

position 配合 top/right/bottom/left 使用：

  • static（默认）：正常文档流
  • relative：相对定位，相对于自身原位置
  • absolute：绝对定位，相对于最近的已定位祖先
  • fixed：固定定位，相对于浏览器窗口
  • sticky：粘性定位，滚动时"粘"在某个位置
👇 relative 定位演示
正常位置
relative 定位（top:10px, left:20px）
正常位置
/* 相对定位 */ .box { position: relative; top: 10px; left: 20px; } /* 绝对定位 */ .box { position: absolute; top: 0; right: 0; } /* 固定定位（导航栏常用） */ .nav { position: fixed; top: 0; left: 0; width: 100%; }

3. z-index（层叠顺序）

z-index 控制定位元素的堆叠顺序，值越大越靠前。

注意：只有 position 不为 static 时才生效。

👇 z-index 对比
z-index: 1
z-index: 2
z-index: 3
/* z-index 必须配合 position 使用 */ .box1 { position: absolute; z-index: 1; /* 最底层 */ } .box2 { position: absolute; z-index: 2; /* 中间层 */ } .box3 { position: absolute; z-index: 3; /* 最顶层 */ }

4. visibility（可见性）

visibility 控制元素是否可见，但保留占据的空间。

与 display: none 的区别：visibility 隐藏后仍占位。

👇 visibility vs display:none
正常显示
visibility: hidden（隐藏但占位）
正常显示
/* 隐藏但占位 */ .box { visibility: hidden; } /* 完全隐藏不占位 */ .box { display: none; }

5. float（浮动）

float 让元素脱离文档流，向左或向右浮动。

现在多用 Flex/Grid 替代，但了解它有助于理解历史布局。

👇 float 演示
float: left
float: left
float: left
⚠️ 清除浮动：浮动元素父容器需要清除浮动，否则高度塌陷。
/* 左浮动 */ .box { float: left; } /* 右浮动 */ .box { float: right; } /* 清除浮动（推荐） */ .parent::after { content: ""; display: block; clear: both; } /* 或者父容器设置 overflow */ .parent { overflow: hidden; }
