🔲 CSS Grid布局

1. Grid 基础概念

Grid 是二维布局的王牌，可以同时控制行和列。

  • 容器：设置 display: grid 的元素
  • 项目：容器的直接子元素
  • 行：水平方向的轨道
  • 列：垂直方向的轨道
  • 单元格：行和列交叉形成的区域

/* 容器设置 */ .container { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 10px; } /* fr 单位表示比例，1fr 1fr 1fr 表示三等分 */

5. 跨列与跨行

grid-column 和 grid-row 让项目跨越多个单元格。

💡 Grid vs Flex：Grid 适合二维布局（行+列），Flex 适合一维布局（行或列）。实际开发中可以结合使用。
