📦 CSS盒模型

1. 盒模型概念

每个HTML元素都可以看作一个矩形盒子，由四部分组成：

  • content：内容区域（文字、图片等）
  • padding：内边距（内容与边框之间的距离）
  • border：边框
  • margin：外边距（元素与其他元素之间的距离）
👇 盒模型可视化演示
margin（外边距）
border（边框）
padding（内边距）
content（内容）

2. width / height

width 和 height 设置的是内容区域的宽高，不包含 padding 和 border。

👇 固定宽度 vs 最大宽度
固定宽度 300px
最大宽度 300px（可缩小）
<div style="width: 300px;">固定宽度 300px</div> <div style="max-width: 300px;">最大宽度 300px</div>

3. box-sizing

box-sizing 控制盒模型的计算方式：

  • content-box（默认）：width/height 只包含内容
  • border-box：width/height 包含 content + padding + border

最佳实践：全局设置 * { box-sizing: border-box; }

👇 box-sizing 对比演示

content-box（默认）

实际宽度：200 + 20×2 + 5×2 = 250px

border-box

实际宽度：200px（包含padding和border）
/* 全局设置 border-box */ * { box-sizing: border-box; } /* content-box（默认） */ .box1 { width: 200px; padding: 20px; border: 5px solid #3498db; /* 实际宽度 = 200 + 20×2 + 5×2 = 250px */ } /* border-box */ .box2 { width: 200px; padding: 20px; border: 5px solid #e74c3c; box-sizing: border-box; /* 实际宽度 = 200px（包含padding和border） */ }

4. padding（内边距）

padding 控制内容与边框之间的距离。

👇 padding 四个方向
padding: 20px
四边都是20px
padding: 10px 20px
上下10px 左右20px
padding: 10px 20px 30px
上10 左右20 下30
padding: 10px 20px 30px 40px
上 右 下 左
/* 四个方向相同 */ padding: 20px; /* 上下 | 左右 */ padding: 10px 20px; /* 上 | 左右 | 下 */ padding: 10px 20px 30px; /* 上 | 右 | 下 | 左（顺时针） */ padding: 10px 20px 30px 40px; /* 单独设置 */ padding-top: 10px; padding-right: 20px; padding-bottom: 30px; padding-left: 40px;

5. margin（外边距）

margin 控制元素与其他元素之间的距离。

注意：margin 可以设置负值，让元素向反方向移动。

👇 margin 演示
margin: 15px
margin: 15px
💡 margin 居中技巧：设置 margin: 0 auto; 可以让块级元素水平居中。
/* 水平居中 */ margin: 0 auto; /* 负margin */ margin-top: -20px; /* 向上移动 */ /* margin 塌陷问题 */ /* 父子元素的垂直margin会合并，取较大值 */ .parent { margin-top: 20px; } .child { margin-top: 30px; /* 实际父元素顶部margin = max(20, 30) = 30px */ }

6. border（边框）

border 设置元素边框，常用简写：border: 宽度 样式 颜色

👇 不同边框样式
solid 实线
dashed 虚线
dotted 点线
double 双线
/* 简写 */ border: 2px solid #3498db; /* 单独设置 */ border-width: 2px; border-style: solid; border-color: #3498db; /* 单边设置 */ border-top: 2px solid #e74c3c; border-bottom: 1px dashed #2ecc71; /* 无边框 */ border: none;

7. border-radius（圆角）

border-radius 设置圆角，值可以是 px 或 %。

设置 border-radius: 50% 可以变成圆形。

👇 不同圆角效果
10px
20px
50%
胶囊形
/* 四角相同 */ border-radius: 10px; /* 圆形（正方形元素） */ border-radius: 50%; /* 胶囊形（长方形元素） */ border-radius: 30px; /* 单独设置四角 */ border-radius: 10px 20px 30px 40px; /* 左上 右上 右下 左下 */

8. overflow（溢出处理）

overflow 控制内容超出容器时的处理方式。

👇 overflow 不同值对比

overflow: visible

这是一段很长的文字内容，会溢出容器显示在外面

overflow: hidden

这是一段很长的文字内容，超出部分会被隐藏

overflow: auto

这是一段很长的文字内容，超出部分会出现滚动条
/* 默认，溢出内容可见 */ overflow: visible; /* 溢出内容隐藏 */ overflow: hidden; /* 溢出内容显示滚动条 */ overflow: auto; /* 只显示横向滚动条 */ overflow-x: auto; /* 只显示纵向滚动条 */ overflow-y: auto;
