✨ CSS动效与变换

1. transition（过渡）

transition 实现属性的平滑过渡，最简单的动效方式。

语法：transition: 属性 时长 缓动函数;

👇 鼠标悬停查看过渡效果
颜色过渡
缩放

/* 基础过渡 */ .box { transition: all 0.3s ease; } /* 悬停效果 */ .box:hover { background: #e74c3c; transform: scale(1.1); } /* 单个属性过渡 */ .box { transition: transform 0.3s ease; } /* 多个属性分别设置 */ .box { transition: transform 0.3s ease, box-shadow 0.5s ease; }

2. transform（变换）

transform 对元素进行2D或3D变换，不会影响其他元素。

常用变换：rotate、scale、translate、skew

👇 不同变换效果

/* 旋转 */ transform: rotate(45deg); /* 缩放 */ transform: scale(1.5); /* 放大1.5倍 */ transform: scale(0.8); /* 缩小 */ /* 移动 */ transform: translate(20px, 10px); /* 倾斜 */ transform: skew(10deg); /* 组合多个变换 */ transform: rotate(45deg) scale(1.2); /* 原点 */ transform-origin: center center;

3. animation（动画）

animation 配合 @keyframes 实现复杂动画。

比 transition 更强大，可以定义多个关键帧。

/* 定义动画 */ @keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } @keyframes spin { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-20px); } } /* 使用动画 */ .box { animation: pulse 1.5s infinite; } /* 动画属性 */ animation: 名称 时长 缓动函数 延迟 次数 方向 填充模式; /* 常用值 */ animation: spin 2s linear infinite; animation: bounce 1s ease-in-out infinite alternate;
💡 性能提示：
• 只用 transform 和 opacity 做动画，性能最好
• 避免用 width、height、margin 做动画，会触发重排
• will-change 属性可以提前告知浏览器优化
