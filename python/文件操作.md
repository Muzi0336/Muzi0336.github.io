# 文件操作

**一、文件操作示例**

**1.1 保存学生信息**
```python
def save_to_file(student_list, filename="student_info.txt"):
    with open(filename, "w", encoding="utf-8") as f:
        for stu in student_list:
            f.write(f"姓名：{stu.name}，学号：{stu.student_id}\n")
    print("保存成功")
```

**要点**
> - `with open(...) as f`：**自动打开并关闭文件**，安全不报错
> - `"w"`：写入模式，**覆盖原有内容**
> - `encoding="utf-8"`：支持中文
> - `f.write(...)`：向文件写入一行文本
> - `\n`：换行，让每个学生占一行

---
**1.2 读取文件内容**
```python
def read_file(filename="student_info.txt"):
    try:
        with open(filename, "r", encoding="utf-8") as f:
            print("文件内容：")
            print(f.read())
    except FileNotFoundError:
        print("文件不存在")
```
**要点**
> - `"r"`：只读模式
> - `f.read()`：一次性**读取全部内容**
> - `try ... except`：捕获文件不存在错误，程序不崩溃
> - 不用手动 `close()`，`with` 会自动处理

---
**二、文件操作核心概念**

**2.1 文件路径**

> - 相对路径：`student_info.txt`（当前文件夹）
> - 绝对路径：`C:/data/student.txt`

**2.2 编码**

> - 必须加 `encoding="utf-8"` 才能正常读写中文，否则会乱码。

**2.3 文件打开模式**
    
> - `r`：只读（默认），文件不存在会报错
> - `w`：只写，文件不存在则创建，存在则**覆盖清空**
> - `a`：追加，文件不存在则创建，存在则在末尾添加
> - `r+`：可读可写
    

---
**三、open() 函数基础**
```python
f = open("文件路径", "模式", encoding="utf-8")
f.close()
```
缺点：
- 必须手动 `close()`，否则会占用文件、数据可能写不进去。
- 出错时可能来不及关闭。

---
**四、with 语句（最推荐写法）**
**自动打开 + 自动关闭文件**，安全、简洁、不会漏关。
```python
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
```
执行完缩进代码块后，**文件自动关闭**，无需 `f.close()`。

---
**五、读文件操作**

**5.1 一次性读取全部内容**
```python
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
print(content)
```

**5.2 按行读取（适合大文件）**
```python
with open("test.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())  # strip() 去掉换行、空格
```

**5.3 读取所有行到列表**
```python
with open("test.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()

print(lines)  # ['第一行\n', '第二行\n', ...]
```

---
**六、写文件操作**

**6.1 覆盖写入（w 模式）**
原有内容会被清空。
```python
with open("test.txt", "w", encoding="utf-8") as f:
    f.write("第一行内容\n")
    f.write("第二行内容\n")
```
**6.2 追加写入（a 模式）**
不会覆盖，在文件末尾添加。
```python
with open("test.txt", "a", encoding="utf-8") as f:
    f.write("追加的一行\n")
```

---
**七、异常处理（重要）**

文件不存在、权限不足时会报错，用 `try-except` 捕获。
```python
try:
    with open("test.txt", "r", encoding="utf-8") as f:
        print(f.read())
except FileNotFoundError:
    print("文件不存在")
except PermissionError:
    print("没有权限访问该文件")
except Exception as e:
    print("未知错误：", e)
```
