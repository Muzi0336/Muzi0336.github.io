# 迭代相关函数

```python
range(5)             # 0, 1, 2, 3, 4
range(1, 10)         # 1, 2, 3, ..., 9
range(0, 10, 2)      # 0, 2, 4, 6, 8

enumerate(["a", "b", "c"])  # (0, 'a'), (1, 'b'), (2, 'c')
zip([1, 2], ["a", "b"])     # (1, 'a'), (2, 'b')
reversed([1, 2, 3])         # 反转迭代器
sorted([3, 1, 2])           # 排序
```
