# 数学相关函数

```python
abs(-5)              # 绝对值：5
max(1, 2, 3)         # 最大值：3
#max([1, 3, 5, 7, 9])# 最大值：9
min(1, 2, 3)         # 最小值：1
sum([1, 2, 3])       # 求和：6
sum() / len()		 # 平均值
len()				 # 元素数量
#len([1, 3, 5, 7, 9])# 元素数量：5
round(3.14159, 2)    # 四舍五入：3.14
pow(2, 3)            # 幂运算：8
divmod(10, 3)        # 商和余数：(3, 1)
```
- 适用范围：这 5 个函数都支持**列表、元组、字符串**等可迭代对象（字符串会按 ASCII 码比较大小）。
>   max（ ）-> len（ ） 
 - 对空列表求 `sum()` 会返回 0，但求 `min()` / `max()` 会报错，使用前要做非空判断。
