# Python 解包（Unpacking）完全指南

你展示的是**最基本的元组解包**，Python 还提供了更丰富的解包方式。

---

**1. 基础解包（一一对应）**

```python
# 元组解包
t1 = (11, 22, 33, 44)
a, b, c, d = t1
print(a, b, c, d)  # 11 22 33 44

# 列表解包
lst = [11, 22, 33, 44]
a, b, c, d = lst
print(a, b, c, d)  # 11 22 33 44

# 字符串解包
s = "ABCD"
a, b, c, d = s
print(a, b, c, d)  # A B C D

# 集合解包（注意无序）
set1 = {11, 22, 33, 44}
a, b, c, d = set1  # 可以工作，但顺序不确定
```

**必须数量严格匹配：**
```python
t1 = (11, 22, 33, 44)
a, b, c = t1  # ValueError: too many values to unpack
a, b, c, d, e = t1  # ValueError: not enough values
```

---

**2. 使用 `*` 收集多余元素**

**收集到列表**

```python
# 收集剩余元素
t1 = (11, 22, 33, 44, 55)
a, b, *rest = t1
print(a)     # 11
print(b)     # 22
print(rest)  # [33, 44, 55]

# 收集中间元素
a, *middle, z = t1
print(a)       # 11
print(middle)  # [22, 33, 44]
print(z)       # 55

# 收集到末尾
*start, last = t1
print(start)  # [11, 22, 33, 44]
print(last)   # 55

# 可以收集空列表
a, b, c, *rest = (11, 22, 33)
print(rest)  # [] 空列表
```

**多个 `*` 不允许**

```python
# 错误示例
a, *b, *c = (1, 2, 3, 4)  # SyntaxError: multiple starred expressions
```

---

**3. 嵌套解包**

```python
# 嵌套元组
t1 = (11, 22, (33, 44))
a, b, (c, d) = t1
print(a, b, c, d)  # 11 22 33 44

# 复杂嵌套
data = (1, 2, (3, 4, (5, 6)))
a, b, (c, d, (e, f)) = data
print(a, b, c, d, e, f)  # 1 2 3 4 5 6

# 列表与元组混合
mixed = [1, 2, (3, 4)]
x, y, (z, w) = mixed
print(x, y, z, w)  # 1 2 3 4
```

---

**4. `*` 在函数调用中解包**

**解包参数列表**

```python
def add(a, b, c):
    return a + b + c

# 列表/元组解包
args = (10, 20, 30)
print(add(*args))  # 60

# 字典解包（使用 **）
kwargs = {'a': 10, 'b': 20, 'c': 30}
print(add(**kwargs))  # 60

# 混合使用
args = (10, 20)
print(add(*args, c=30))  # 60
```

**解包到函数参数**

```python
def func(a, b, *args, **kwargs):
    print(f"a={a}, b={b}")
    print(f"args={args}")
    print(f"kwargs={kwargs}")

func(1, 2, 3, 4, 5, x=10, y=20)
# a=1, b=2
# args=(3, 4, 5)
# kwargs={'x': 10, 'y': 20}
```

**5. 列表/元组拼接与扩展**

```python
# 拼接列表
list1 = [1, 2, 3]
list2 = [4, 5, 6]
combined = [*list1, *list2]
print(combined)  # [1, 2, 3, 4, 5, 6]

# 拼接元组
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
combined = (*tuple1, *tuple2)
print(combined)  # (1, 2, 3, 4, 5, 6)

# 混合类型
mixed = [*list1, *tuple2, 7, 8]
print(mixed)  # [1, 2, 3, 4, 5, 6, 7, 8]
```

---

**6. 字典解包**

```python
# 合并字典
dict1 = {'a': 1, 'b': 2}
dict2 = {'c': 3, 'd': 4}
merged = {**dict1, **dict2}
print(merged)  # {'a': 1, 'b': 2, 'c': 3, 'd': 4}

# 覆盖值（后面的覆盖前面的）
dict1 = {'a': 1, 'b': 2}
dict2 = {'b': 100, 'c': 3}
merged = {**dict1, **dict2}
print(merged)  # {'a': 1, 'b': 100, 'c': 3}

# 添加新键值
base = {'x': 1, 'y': 2}
new = {**base, 'z': 3}
print(new)  # {'x': 1, 'y': 2, 'z': 3}
```

**7. 循环中的解包**

```python
# 遍历元组列表
points = [(1, 2), (3, 4), (5, 6)]
for x, y in points:
    print(f"x={x}, y={y}")

# 使用 enumerate
for index, (x, y) in enumerate(points):
    print(f"索引{index}: ({x}, {y})")

# 遍历字典
person = {'name': 'Alice', 'age': 30, 'city': 'Beijing'}
for key, value in person.items():
    print(f"{key}: {value}")

# 带条件的解包
data = [(1, 'a'), (2, 'b'), (3, 'c')]
for num, char in data:
    if num > 1:
        print(char)  # b, c
```

**8. 交换变量（经典应用）**

```python
# 传统方式需要临时变量
a = 10
b = 20
temp = a
a = b
b = temp

# Python 解包交换（优雅）
a, b = 10, 20
a, b = b, a
print(a, b)  # 20 10

# 交换多个变量
a, b, c = 1, 2, 3
a, b, c = c, a, b
print(a, b, c)  # 3 1 2
```

---

**9. 忽略不需要的值**

```python
# 使用下划线忽略
t1 = (11, 22, 33, 44)
a, _, c, _ = t1
print(a, c)  # 11 33

# 使用 * 忽略剩余
a, b, *_ = t1
print(a, b)  # 11 22

# 忽略多个特定位置
first, *_, last = t1
print(first, last)  # 11 44

# 经典应用：只取第一个和最后一个
scores = [85, 92, 78, 90, 88]
first, *middle, last = scores
print(f"首: {first}, 末: {last}")  # 首: 85, 末: 88
```

**10. 高级应用**

**函数返回多个值的解包**
```python
def get_user_info():
    return "Alice", 25, "alice@example.com"

name, age, email = get_user_info()
print(name, age, email)  # Alice 25 alice@example.com
```

**嵌套结构解包**
```python
# 复杂数据结构
data = {
    'user': ('Bob', 30),
    'scores': [85, 90, 95],
    'meta': {'created': '2024-01-01'}
}

# 解包
(user_name, user_age), (score1, score2, score3), meta_info = data.values()
print(user_name, user_age)  # Bob 30
```

**条件解包（Python 3.10+）**
```python
# 使用 match-case 解包
def process_point(point):
    match point:
        case (x, y):
            print(f"二维点: ({x}, {y})")
        case (x, y, z):
            print(f"三维点: ({x}, {y}, {z})")
        case _:
            print("未知格式")

process_point((10, 20))        # 二维点: (10, 20)
process_point((1, 2, 3))       # 三维点: (1, 2, 3)
```

**常见错误与注意事项**

```python
# 1. 数量不匹配
t = (1, 2)
a, b, c = t  # ValueError

# 2. 不能对生成器多次解包
gen = (x for x in range(3))
a, b, c = gen  # 可以
d, e, f = gen  # StopIteration（已耗尽）

# 3. 集合解包顺序不确定
s = {3, 1, 2}
a, b, c = s  # a可能是1,2,3中的任意一个

# 4. 不能解包 None
x = None
a, b = x  # TypeError

# 5. 星号表达式不能单独使用
*rest = (1, 2, 3)  # SyntaxError
```

---

**总结表**

| 场景 | 语法示例 | 结果 |
|------|---------|------|
| 基础解包 | `a, b, c = (1, 2, 3)` | `a=1, b=2, c=3` |
| 收集剩余 | `a, *b = (1, 2, 3)` | `a=1, b=[2, 3]` |
| 收集中间 | `a, *b, c = (1,2,3,4)` | `a=1, b=[2,3], c=4` |
| 忽略值 | `a, _, c = (1,2,3)` | `a=1, c=3` |
| 交换变量 | `a, b = b, a` | 交换值 |
| 合并列表 | `[*a, *b]` | 新列表 |
| 合并字典 | `{**d1, **d2}` | 新字典 |
| 函数参数 | `func(*args)` | 解包为参数 |

**核心原则：** 解包让代码更 Pythonic，但要注意数量匹配和可读性。
