# 装饰器

**一、装饰器本质：闭包 + 高阶函数**
```python
import functools

def my_decorator(func):
    @functools.wraps(func)  # 保留原函数名字、注释
    def wrapper(*args, **kwargs):
        print("函数执行前")
        result = func(*args, **kwargs)  # 调用原函数
        print("函数执行后")
        return result
    return wrapper

@my_decorator
def say_hello(name):
    print(f"Hello, {name}!")

say_hello("沐子")
```

执行顺序：
1. 定义 `my_decorator`
2. `@my_decorator` → `say_hello = my_decorator(say_hello)`
3. 调用 `say_hello("沐子")` → 实际调用 `wrapper`
4. 前打印 → 原函数 → 后打印 → 返回结果

作用：**不修改原函数，给函数加功能**

**二、带参数的装饰器（三层嵌套）**
```python
def repeat(times):
    def decorator(func):
        @functools.wraps(func)
        def wrapper(*args, **kwargs):
            for _ in range(times):
                result = func(*args, **kwargs)
            return result
        return wrapper
    return decorator

@repeat(3)
def say_hello(name):
    print(f"Hello, {name}!")

say_hello("沐子")
```

结构：
- 第一层 `repeat(times)`：接收参数
- 第二层 `decorator(func)`：接收函数
- 第三层 `wrapper`：真正执行
执行：函数会被**重复执行 3 次**

**三、类装饰器**
```python
class Counter:
    def __init__(self, func):
        self.func = func
        self.count = 0
        functools.update_wrapper(self, func)

    def __call__(self, *args, **kwargs):
        self.count += 1
        print(f"调用次数: {self.count}")
        return self.func(*args, **kwargs)

@Counter
def add(a, b):
    return a + b

add(1,2)
add(3,4)
```

原理：
- `@Counter` 等价于 `add = Counter(add)`
- 调用 `add()` 时，实际调用 `__call__`
- 可以**保存状态（count）**，函数装饰器做不到

---
**四、多个装饰器嵌套顺序**
```python
@decorator_a
@decorator_b
def func():
    pass
```

执行顺序：
1. 进入 decorator_a
2. 进入 decorator_b
3. 执行 func
4. 从 decorator_b 返回
5. 从 decorator_a 返回
口诀：**从上到下进入，从下到上返回**

**四个实战练习（直接可运行）**

**练习 1：计时装饰器**
```python
import time
def timer(func):
    @functools.wraps(func)
    def wrapper(*args,**kwargs):
        t1 = time.time()
        res = func(*args,**kwargs)
        t2 = time.time()
        print(f"耗时：{t2-t1:.4f}s")
        return res
    return wrapper
```

**练习 2：参数必须为正数**
```python
def positive_only(func):
    @functools.wraps(func)
    def wrapper(a):
        if a < 0:
            raise ValueError("必须正数")
        return func(a)
    return wrapper
```

**练习 3：日志装饰器**
```python
def log(func):
    @functools.wraps(func)
    def wrapper(*args,**kwargs):
        print(f"调用 {func.__name__}, 参数：{args}")
        return func(*args,**kwargs)
    return wrapper
```

**练习 4：带参数重试装饰器（进阶）**
```python
def retry(times=3):
    def decorator(func):
        @functools.wraps(func)
        def wrapper(*args,**kwargs):
            for i in range(times):
                try:
                    return func(*args,**kwargs)
                except Exception as e:
                    print(f"第{i+1}次失败")
        return wrapper
    return decorator
```

**极简总结（直接复制进 Obsidian）**
- 装饰器 = 高阶函数 + 闭包
- `@xxx` 是语法糖
- `functools.wraps` 保留原函数信息
- 带参数装饰器：三层嵌套
- 类装饰器：用 `__call__`，可保存状态
- 多个装饰器：从上到下进，从下到上出
